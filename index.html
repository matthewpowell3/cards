HTML (Hypertext Markup Language) is a markup language used to create the structure and content of web pages. It uses various tags and elements to define the different parts of a web page such as headings, paragraphs, images, links, and more.

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of HTML documents. It allows web developers to style the layout, typography, colors, and other visual aspects of a web page, separate from the HTML content.

JavaScript is a programming language used to create interactive and dynamic web content. It can be used to add interactivity to web pages, create animations, validate forms, and more.interactivity to web pages, create animations.
